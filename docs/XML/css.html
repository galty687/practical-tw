CSS 基本介绍¶

什么是CSS¶

样式语言用于内容的格式呈现。

CSS语法¶

Selector {property1: value1; property2: value2}

常见写法

selector {
  property1: value1;
  property2: value2;
}

例如：

h1 {
	color: blue;
}

选择类

<h1 class="big-header">Title</h1>
<a href="index.html">Home</a>

.class-name {property: value}

<button class="btn btn-1">
  button 1
</button>

<button class="btn btn-2">
  button 2
</button>

<button class="btn btn-3">
  button 3
</button>

  padding: 10px;

  property: value;

	property: value;

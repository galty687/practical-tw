=========================
教材写作风格指南
=========================

（不断更新中）

格式
===================

文件名
-------------

1. 文件名命名时，采用标题的英译，单词之间使用”-“连接，文件名中不得包含空格。


    .. example::
    
        “虚拟角色的创建”的topic，命名为 `the-creation-of-persona.dita` 
    

图片
-------------

1. 截图。在截图时，考虑全屏和局部；
2. 尺寸。待定
3. 存放路径。图片统一放入 `images` 文件夹
4. 图片命名时，采取图片 `Caption` 英译，规则同上方文件名命名规则。
